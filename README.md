## Featured Project — RDL (Dental Lab Order Management Platform)

A production-grade platform that replaces paper-based workflows in dental laboratories with a centralized digital system.

**Stack:** Node.js · Express · MongoDB · React · Expo (React Native) · Cloudinary · Railway

**Key highlights:**
- Order lifecycle tracking — from intake to delivery (pending → in-progress → delivered)
- Role-based access control (admin and employee roles)
- OTP + PIN login with JWT and refresh token rotation
- Secure file attachments (X-rays, STL files) via signed Cloudinary URLs — raw URLs never exposed
- Brute-force lockout on OTP and PIN attempts, CORS allow-listing, regex injection protection
- Printable Work Order documents, payment tracking, patient and doctor management
- Activity log tracking all significant actions across the system
- React web dashboard + Expo mobile app for on-the-go access
